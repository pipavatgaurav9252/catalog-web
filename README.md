# 📸 Catalog-Z Website Clone

A responsive photo gallery website built with **HTML**, **CSS**, and **JavaScript**. This project recreates the Catalog-Z design, featuring a modern layout, image gallery, search bar, responsive navigation menu, and footer.

---

## 🚀 Features

- ✅ Responsive Header with Navigation
- ✅ Mobile Menu Toggle
- ✅ Hero Section with Search Bar
- ✅ Latest Images Gallery
- ✅ Image Cards with Date & View Count
- ✅ Pagination Buttons
- ✅ Responsive Footer
- ✅ Social Media Icons
- ✅ Clean and Modern UI
- ✅ External CSS Styling
- ✅ JavaScript Navigation Toggle

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
